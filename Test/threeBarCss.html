三栏布局

1. 浮动+margin

结构：left-> right -> mid
.method1-left {
 float: left;
}
.method1-right {
 float: right;
}

.method1-mid {
 margin-left: 205px;
 margin-right: 155px;
}

注意！：当“非float的元素”和“float的元素”在一起的时候,如果非float元素在先,那么float的元素将受到排斥。
所以mid div要放在最下面

2. 浮动+BFC

结构：left-> right -> mid
.method2-left {
 float: left;
}
.method2-right {
 float: right;
}
可以发现 有margin但因为BFC折叠了
.method1-mid {
 overflow: hidden;
}

注意！：同上，当“非float的元素”和“float的元素”在一起的时候,如果非float元素在先,那么float的元素将受到排斥。
所以mid div要放在最下面

3. flex

结构：left-> mid -> right
.method3-container {
 display: flex;
}
.method3-left {}
.method3-mid {
 flex: 1;
}
.method3-right { }

4. 父相子绝

结构：left-> right -> mid
.method4-container {
 position: relative;
}
.method4-left {
 position: absolute;
 left: 0;
}
.method4-right {
 position: absolute;
 right: 0;
}
.method4-mid {
 margin: 0 155px 0 205px;
}
注意！：当“非float的元素”和“float的元素”在一起的时候,如果非float元素在先,那么float的元素将受到排斥。
所以mid div要放在最下面

5. 圣杯【三栏都使用float】

结构：mid -> left-> right
.sub-container {
 margin-left: left;
 width: 100%;
}
.method5-mid {
 float: left;
 width: 100%;
}
注意！：mid div要放在最上面 贼麻烦算了吧
.method5-left {
 float: left;
 margin-left: -100%;
 left: -205px;
 position: relative;
}
.method5-right {
 float: left;
 margin-left: -155px;
 right: -150px;
 position: relative;
}

6. 双飞翼

结构：mid -> left-> right
.method6-mid {
 float: left;
 width: 100%;
}
.mid-content {
 margin-left: 200px;

 height: 100%;
}

}
