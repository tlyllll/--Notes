两栏布局

1. 浮动+margin

.method1-left {
 float: left;
}
.method1-right {
 margin-left: 205px;
}

可以看出，两个方块之间的间隙是由右边方块的margin-left决定的

2. 浮动+BFC

.method2-left {
 float: left;
}
.method2-right {
 overflow: hidden;
}
可以发现 有margin但因为BFC折叠了

3. flex

.method3-container {
 display: flex;
}
.method3-left {}
.method3-right {
 flex: 1;
}
左右margin没折叠

4. 父相子绝

.method4-container {
 position: relative;
}
.method4-left {
 position: absolute;
}
.method4-right {
 position: absolute;
 left: 200px;
 right: 0;
}
right 的左右必须都设置，如果right:0不设置的话div撑不开
中间没有间隙因为right的left限死了200px
